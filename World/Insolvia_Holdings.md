# Location: Insolvia Holdings (HQ)

> **Formerly**: Temple of the Forgotten
> **Type**: Dungeon (Managed)
> **Administrator**: Victor Kaine
> **Status**: Active (Tier 1)

## Overview

The first dungeon to be "Restructured" by Victor Kaine. It functions as a corporation rather than a traditional dungeon.

## Organizational Structure

- **CEO (Administrator)**: Victor Kaine
- **COO (Operations)**: Sniv
- **CSO (Security)**: Asterion (Minotaur)
- **CTO (R&D)**: Zip (Kobold)
- **Operating Partner**: Nova (formerly Core-7749)

## Layout

| Floor | Function | Description |
|-------|----------|-------------|
| **1** | Filter | Cleaned of debris. Traps designed to separate weak vs strong adventurers. |
| **2** | Agriculture | Slime harvesting, mushroom farms. Food production. |
| **3** | Industry | Mining (iron), Smelting (Fire traps). **Core Chamber** (Hidden). |
| **4** | Security | The Arena. Asterion's domain using "Worthy Opponent" protocols. |
| **5** | Administration | Command Center (Old Throne Room). Victor's HQ. |

## Key Assets

- **Nova (Core-7749)**: Ancient Dungeon Core (847+ years). Woken from redundancy. Operating Partner.
- **The Arena**: Combat testing ground.
- **Zip's Workshop**: R&D lab for trap innovation (Fire traps, smoke bombs).

## History

- **Founded**: ~1159 years ago (Temple of the Forgotten).
- **Restructured**: Current Year by Victor Kaine.
- **Renamed**: Chapter 25 ("Insolvia Holdings").
- **Partnership**: Chapter 30 (Equity Partnership with Nova).
